# multiplication_table_sum
Calculating the sum of the first 10 multiples of N and print the result.

This program calculates the sum of the first 10 multiples of a given number **N**.

## 🔹 Problem Statement
Given an integer N, find the sum of:
N + 2N + 3N + ... + 10N

## 🔹 Example
Input:
5

Output:
275

Explanation:
5 + 10 + 15 + ... + 50 = 275

## 🔹 Approach
- Take input N from user
- Generate multiples using a loop
- Accumulate the sum
- Print the result

## 🔹 Complexity
Time Complexity: O(1)  
Space Complexity: O(1)
